# 202275440-KawtharAlomran-assignment2
## Project Description

This project is a personal portfolio website built using HTML, CSS, and JavaScript.  
It includes three main sections: About Me, Projects, and Contact, along with a dynamic greeting message based on the current time.
The project focuses on practicing core web development concepts and building a responsive layout that works on different devices.

## Setup Instructions

To run the project locally:

1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in your browser.
