# AI Usage Report

## Tools Used & Use Cases
During this assignment, ChatGPT was used as an AI-assisted tool.

It was used for:
- Providing explanations for CSS concepts that were initially unclear, particularly layout behavior and sizing.
- Debugging responsive layout issues, especially overflow and width behavior.
- Improving code structure and overall readability.
- Reviewing and refining written comments.
- Suggesting the structure and required components of documentation files.
- Enhancing the organization and clarity of documentation content.
- Exploring and refining UI styling decisions, including the greeting message and navigation, by iteratively evaluating alternative design approaches before selecting the final implementation.


AI assistance was primarily used as a learning and support tool rather than as a direct code generator.

## Benefits & Challenges

Using AI tool provided several benefits, including:

- Faster debugging and problem identification.
- Guidance on structuring documentation properly.
- Improved the clarity and quality of code comments.
- Clear explanations of different technical concepts.

Although AI was helpful, there were some limitations:

- AI sometimes provided solutions that were more complex than necessary for the assignment requirements.
- Some responses required follow-up questions to obtain more precise guidance, as the initial answers were sometimes too general.
- Some responses did not fully align with the intended request and required additional clarification.


## Learning Outcomes

Through the use of AI in this assignment, I learned:
- A stronger understanding of how CSS properties such as width, padding, and box-sizing interact.
- How to properly debug layout overflow issues.
- How Flexbox behaves across different screen sizes.
- The importance of reviewing and validating AI-generated suggestions.
- How to structure documentation clearly and professionally.
- The importance of asking clear and specific technical questions to receive accurate guidance.


## Responsible Use & Modifications

All AI-generated suggestions were carefully reviewed, tested, and adjusted to ensure correctness and alignment with the assignment requirements.  

The final implementation reflects my personal understanding of the concepts applied in the project.
